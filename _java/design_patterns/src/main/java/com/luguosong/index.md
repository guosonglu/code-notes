# 定义

`模式`是在特定环境下人们解决某类重复出现问题的一套成功或有效的解决方案

`设计模式`是在特定环境下为了解决某一通用软件设计问题提供的一套定制的解决方案，该方案描述了对象和类之间的相互作用。
